# Carpeta `src/styles/` - Hojas de Estilo CSS Personalizadas

Este directorio agrupa los archivos de estilos CSS específicos. Complementan el uso generalizado de **TailwindCSS** en el portal, añadiendo animaciones complejas, transiciones, y estilos personalizados para componentes de terceros o específicos.

- **[`carousel.css`](file:///c:/Users/Usuario/OneDrive/Documentos/GitHub/Colejus/colejus-frontend-main/src/styles/carousel.css):**
  Estilos y animaciones de transición (desvanecimiento y deslizamientos) para el carrusel de novedades y noticias en la Home.
- **[`courtcarousel.css`](file:///c:/Users/Usuario/OneDrive/Documentos/GitHub/Colejus/colejus-frontend-main/src/styles/courtcarousel.css):**
  Ajustes de espaciados, bordes redondeados y efectos hover específicos para el carrusel de juzgados y dependencias.
- **[`derechofijo.css`](file:///c:/Users/Usuario/OneDrive/Documentos/GitHub/Colejus/colejus-frontend-main/src/styles/derechofijo.css):**
  Estilos del formulario de solicitud y cálculo del Derecho Fijo. Contiene reglas multimedia específicas para mejorar el diseño impreso (las boletas generadas).
- **[`navbar.css`](file:///c:/Users/Usuario/OneDrive/Documentos/GitHub/Colejus/colejus-frontend-main/src/styles/navbar.css):**
